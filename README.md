# calculadora-py

`by Hapistello Gnosko`

Calculadora de terminal em Python com as quatro operações básicas, validação de entrada e loop de uso continuável. Projeto fundacional — onde a lógica começou.

## `// usage`

```bash
git clone https://github.com/HapistelloGnosko/calculadora-py
cd calculadora-py
python calculadora.py
```

---

## `// features`

- Operações: adição, subtração, multiplicação, divisão
- Tratamento de divisão por zero
- Validação de entrada numérica
- Loop com opção de continuar ou sair
